# AGENTS.md

## ドキュメント

- 思考および開発ドキュメントは英語で記述すること。
- 英語で開発ドキュメントを作成した後、リポジトリルートからのパス構造を模倣した日本語翻訳ファイルを .docs/ja/ ディレクトリに作成すること。
- 英語版と日本語版のドキュメントは、可能な限り迅速に同期すること。

## Git

- コードは適切な小さな単位に分割し、必要に応じて継続的にブランチ作成、コミット、プッシュを行うこと。

## タスクリスト記号 / チェックリスト記号

- 未着手のタスクには `- [ ]` を使用すること。
- 完了済みのタスクには `- [x]` を使用すること。
- 保留中のタスクには `- [-]` を使用すること。
